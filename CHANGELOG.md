# Changelog

## v1.0.5.5 - 2026-05-18

- 更新阿里云公网入口到 `59.110.166.166`，同步固件镜像上报、控制拉取、README 和部署说明。
- 修复网页阈值保存时被实时刷新覆盖的问题，改为记录本次修改草稿并增量保存。
- 延长地震、SOS 等强提醒显示时间，确保高危提醒至少保持 4 秒可见。
- 优化窗帘舵机联动，自动模式下可根据环境光状态打开或关闭窗帘。
- 新增 PM2 与 Nginx 部署配置，便于阿里云服务器按 `3001 -> 80` 反向代理运行。

## v1.0.5 - 2026-05-15

- 新增远程阈值下发能力：网页端可调整空气质量、烟雾、一氧化碳、温湿度、暗光、床垫压力和跌倒相关阈值，固件端会拉取并应用到实时告警判断。
- 优化云端控制链路：固件镜像上报会携带当前联动开关和阈值，HTTP 镜像与控制拉取加入 1 秒超时，采样、上报和控制轮询调整为更高频率。
- 增强网页后端状态管理：控制开关持久化到 `web/data/controls.json`，阈值合并保存，健康接口返回阈值、联动和设备在线状态。
- 新增设备离线保护：超过离线阈值未收到设备上报时，网页显示离线提示并锁定控制和阈值设置，设备恢复后自动解除。
- 改进前端交互反馈：SSE 新增连接状态事件，控制台实时刷新在线状态、保存成功/失败提示、离线横幅和告警摘要。
- 统一本地演示节奏和服务端口：演示刷新改为 1 秒，Web 默认端口和文档示例统一为 `3001`。
